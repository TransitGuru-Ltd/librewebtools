About LibreWebTools
-------------------
LibreWebTools is a work-in-progress toolbox for creating online interactive
web applications in addition to providing a simple framework to host a basic
website.

Configuration and Dependencies
------------------------------
LibreWebTools depends on Apache 2.2, MariaDB or MySQL 5.5, and PHP 5.5. Later 
versions of each package will also be supported. More information will be
provided on how to configure the tools.
